Konverze a import 3D Scény

Pro úspěšné nahrání je scény tvořené v této verzi enginu je třeba stáhnout následující balíček Unity, který obsahuje potřebné SDK pro převod scény. Po přidání balíčku jej lze použít, aby scéna plně podporovala exponáty podporované tímto systémem.

Je potřeba učinit následující kroky:

  1. Nahradit objekt "hráče" příslušným objektem z SDK.
  2. Přidat požadované popisky do scény pomocí objektu z SDK.
  3. Podobně přidat místa, která modifikují interakci uživatele.

Samotné SDK v tomto procesu pomůže pomocí editorového okna s podrobnějšími informacemi.

Tento engine akktuálně není podporován. Využijte prosím engine Unity.
